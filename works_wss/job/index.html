Job Category

職種紹介

Web Designer Webデザイナー

企業や個人などのクライアントから依頼されたWebサイトのデザインを担当・制作する職種です。クライアントが思い描いているWebサイトを制作できるよう、使い勝手がよく魅力が伝わるようなデザインを制作します。

Web Coder Webコーダー

Webデザイナーが作成したWebデザインを、Webブラウザ上で表示できるように構築する職種です。HTML、CSSなどといった言語を使用してソースコードを記述し、実際にインターネット上で見ることができるようにします。

Front End Engineer フロントエンドエンジニア

WebサイトやWebサービスの「フロントエンド」を設計・構築するエンジニアをさします。JavaScriptを使い動的な処理を行うなど、コーダーよりも幅広く深いスキルが求められます。

Web Director Webディレクター

クライアントと制作サイドの間に立ち、スケジュール管理やディレクションを行い全体を統括します。進行や品質に問題がないかチェックし、プロジェクトが円滑に進むように調整して、チームで目標の達成を目指します。

Employment Support

就職支援

キャリアコンサルティング

訓練受講中に、キャリアコンサルティングを定期的に実施いたします。今後のキャリアプランを共に考え、求人状況など企業情報を共有しながらカウンセリングしていきます。 個々が抱える心配なことや不安なこと、一人ひとりの悩みに寄り添いながら、キャリアの選択肢が広がるよう、「なりたい自分」の実現を目指してサポートいたします。

職業人講話

第一線で活躍している企業の経営者をお招きして直接お話を伺います。 企業によって様々な特色があり、どのような方法で制作をしているかなど、普段聞けないお仕事の内容を知る貴重な機会です。企業の雰囲気を知ることで、実際に働くイメージが沸き、就職意欲がさらに高まる講話となっています。　

制作物の添削・評価

就職活動に必要なポートフォリオや自主的に制作した作品など、デザイン力や技術の向上のため、アドバイスを行います。

模擬面接

生徒同士で模擬面接を行います。ヒアリングを通して、自身がまだ気づけていない特性や価値観をお互いに発見しやすくなります。

アフターフォロー

履歴書・職務経歴書の添削など、訓練中、または訓練終了後もサポートしています。 就職への疑問などありましたら、ご相談ください。
